BUSMASTER Add-Ons

    To enhance specific functionalities in Busmaster Open Source Software, we are offering series of

    commercial add-ons.

    Current development includes the following commercial add-ons.


    FlexRay


    BusMaster FlexRay Add-On supports monitoring, analysis and simulation of FlexRay Bus System.

    FlexRay Add-On v1.0.0 is supported on top of Open Source v3.1.0


    Features Supported: Link


  • Controller Configuration
  • ASAM FIBEX standard v2.0.0, v3.0.0 and v4.0.0
  • Frame Transmission and Monitoring
  • Network Statistics
  • Filters
  • Logging
  • Signal watch
  • Node Simulation

    • Instruments


    BusMaster Instruments Add-On provides intuitive way of displaying and manipulating signal values using

    GUI controls


    Features Supported: Link


  • Instrument Editor
  • Import instruments in BusMaster

    • CAN FD


    BusMaster CAN FD Add-On supports monitoring, analysis and simulation of CAN FD Frames.


    Features Supported: Link


  • Controller Configuration
  • Frame Transmission and Monitoring
  • Network Statistics
  • Filters
  • Logging
  • Signal watch
  • Node Simulation
